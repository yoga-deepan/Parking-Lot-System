# Screenshots Folder

Add your application screenshots here:

1. `dashboard.png` - Dashboard showing available slots
2. `park-vehicle.png` - Park vehicle form with success message
3. `exit-vehicle.png` - Exit vehicle form with payment receipt

## How to Take Screenshots

1. Run the application
2. Open http://localhost:5173 in your browser
3. Take screenshots of each section:
   - Dashboard tab
   - Park Vehicle tab (after parking a vehicle)
   - Exit Vehicle tab (after processing an exit)
4. Save them in this folder with the names above
5. Update the README.md with actual screenshot paths

## Tips for Good Screenshots

- Use a clean browser window (no extra tabs/bookmarks visible)
- Capture the full application interface
- Show successful operations (green success messages)
- Use realistic test data (e.g., "TN01AB1234" for vehicle numbers)
- Consider using a screenshot tool like Snipping Tool or Lightshot
